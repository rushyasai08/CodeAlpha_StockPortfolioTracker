# 📊 Stock Portfolio Tracker (Python)

## 📌 Project Description
This project is a simple Stock Portfolio Tracker built using Python. It allows users to input stock names and quantities, calculates total investment based on predefined stock prices, and displays a complete portfolio summary. The results are also saved to a file for future reference.

---

## 🚀 Features
- Add multiple stocks with quantities  
- Calculates individual and total investment  
- Displays portfolio summary  
- Handles invalid inputs  
- Saves results to a text file (`portfolio.txt`)  

---

## 🛠️ Technologies Used
- Python  
  - Dictionaries  
  - Loops (while, for)  
  - Conditional statements  
  - File handling  
  - User input

## 🎯 How It Works
- User enters stock name (AAPL, TSLA, etc.)  
- Enters quantity  
- Program calculates investment  
- Repeats until user enters "done"  
- Displays final summary and saves to file
